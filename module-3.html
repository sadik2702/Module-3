Module-3: Git

First time you have to follow six steps to push in your github.

  1. initialization: git init
  2. add files to git: git add .
  3. make a commit: git commit -m main
  4. change branch to main: git branch -M main
  5. set url: git remote add origin 'url'
  6. push to git: git push -u origin main

After pushing onece or more you need to follow three steps to push in your git hub.

  1. add all changes: git add .
  2. commit: git commit -m 'text'
  3. git push: git push
